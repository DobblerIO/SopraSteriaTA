# Technical Assessment submission for SopraSteria
This project contains the code submission for the Technical Assessment given by SopraSteria.

## Core Tooling used within the project
 - **React**
 - **Typescript**
 - **Vite**
 - **Playwright**

## Running the Project
Most important is creating a `.env` file in the root of the project and provide an 'API key' for [tomorrow.io](https://www.tomorrow.io/).
There is a `.env.example` file available that can be copied and renamed.

In case you do not have a personal API key, I should have provided my own within our email correspondence.

To run the local development server:
```bash
npm run dev
```

To run the playwright test suite:
```bash
npm run test
```
